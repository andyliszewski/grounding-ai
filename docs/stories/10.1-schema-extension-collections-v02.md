### Collection Name Validation Rules

Valid collection names:
- `science` ✓
- `music-theory` ✓
- `ap-biology` ✓
- `reference-2024` ✓

Invalid collection names:
- `Science` ✗ (uppercase)
- `music_theory` ✗ (underscore)
- `music theory` ✗ (space)
- `-music` ✗ (leading hyphen)
- `music-` ✗ (trailing hyphen)
- `` ✗ (empty)

### Validation Function Signature

```python
# In utils.py
import re

def validate_collection_name(name: str) -> bool:
    """Validate collection name follows kebab-case convention.

    Args:
        name: Collection name to validate

    Returns:
        True if valid kebab-case, False otherwise

    Examples:
        >>> validate_collection_name("science")
        True
        >>> validate_collection_name("music-theory")
        True
        >>> validate_collection_name("Science")
        False
    """
    if not name or not isinstance(name, str):
        return False
    pattern = r'^[a-z0-9]+(-[a-z0-9]+)*$'
    return bool(re.match(pattern, name))
```
