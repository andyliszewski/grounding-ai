# Story 11.1: Corpus Agent Slash Commands

**Epic:** Epic 11 - Interactive Corpus Agents
**Status:** Done
**Owner:** TBD
**Created:** 2026-01-04

---

## Story

**As a** knowledge worker,
**I want** to activate corpus agents via slash commands (e.g., `/ceo`, `/venture-attorney`),
**So that** I get a specialized persona with expertise scoped to the agent's filtered corpus for context-aware assistance.

---

## Acceptance Criteria

1. Corpus agent YAML schema extended with `persona` block (style, expertise, greeting)
2. Each agent in `/agents/` has a corresponding `.claude/commands/<agent>.md` slash command
3. Invoking `/ceo` loads the CEO persona with appropriate communication style
4. Queries while agent is active are scoped to the agent's `corpus_filter` collections
5. Agent personas have distinct personalities appropriate to their professional role
6. `/corpus-agents` command lists all available corpus agents with descriptions
7. Agents can reference their filtered corpus via RAG for grounded responses
8. Agent activation displays a role-appropriate greeting
9. Existing BMAD agents (po, dev, etc.) remain unaffected
10. Generator script creates/updates command files from agent YAMLs
