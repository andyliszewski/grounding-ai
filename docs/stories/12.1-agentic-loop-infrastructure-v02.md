# Story 12.1: Agentic Loop Infrastructure

**Epic:** Epic 12 - Agentic Tool Calling for Local LLMs
**Status:** Done
**Owner:** TBD

## Story

**As a** local LLM user,
**I want** the system to support multi-turn tool calling with an agentic loop,
**So that** the LLM can reason, call tools, receive results, and continue reasoning until it has a complete answer.

---

## Acceptance Criteria

1. Ollama API called with `tools` parameter when agentic mode is enabled
2. Tool calls in LLM response are detected via `tool_calls` array
3. Agent loop continues processing until no more tool calls returned
4. Max iterations safeguard prevents infinite loops (configurable, default: 5)
5. Errors during tool execution are returned as tool result content (not raised as exceptions)
6. Message history correctly accumulates assistant and tool messages
7. Final response extracted and returned to caller
8. Loop handles both single and multiple tool calls per turn

```python
from dataclasses import dataclass
from typing import Protocol, Callable, Any
import json
import requests
import logging

logger = logging.getLogger("agentic")

@dataclass
class AgenticConfig:
    """Configuration for agentic loop behavior."""
    max_iterations: int = 5
    verbose: bool = False
    timeout: int = 120  # seconds per LLM call

@dataclass
class ToolCall:
    """Parsed tool call from LLM response."""
    name: str
    arguments: dict[str, Any]

@dataclass
class AgentLoopResult:
    """Result from running the agentic loop."""
    content: str
    tool_calls_made: list[ToolCall]
    iterations: int
    stopped_reason: str  # "complete", "max_iterations", "error"
```

### Ollama API Integration

```python
def query_ollama_with_tools(
    messages: list[dict],
    tools: list[dict],
    model: str,
    api_url: str,
    timeout: int = 120
) -> dict:
    """
    Send a request to Ollama with tool definitions.

    Returns the raw response dict with 'message' key containing:
    - role: "assistant"
    - content: Optional text response
    - tool_calls: Optional list of tool call requests
    """
    payload = {
        "model": model,
        "messages": messages,
        "tools": tools,
        "stream": False
    }

    response = requests.post(
        api_url,
        json=payload,
        timeout=timeout
    )
    response.raise_for_status()
    return response.json()
```

### Agent Loop Implementation

```python
def run_agentic_loop(
    user_query: str,
    tools: list[dict],
    tool_executor: Callable[[str, dict], str],
    model: str,
    api_url: str,
    config: AgenticConfig = None,
    system_prompt: str = None
) -> AgentLoopResult:
    """
    Run the agentic loop until completion or max iterations.

    Args:
        user_query: The user's question
        tools: List of tool schemas in OpenAI format
        tool_executor: Function to execute tool calls
        model: Ollama model name
        api_url: Ollama API endpoint
        config: Optional configuration
        system_prompt: Optional system message

    Returns:
        AgentLoopResult with final content and metadata
    """
    config = config or AgenticConfig()
    messages = []

    if system_prompt:
        messages.append({"role": "system", "content": system_prompt})

    messages.append({"role": "user", "content": user_query})

    tool_calls_made = []
    iteration = 0

    while iteration < config.max_iterations:
        iteration += 1

        if config.verbose:
            logger.info(f"Agentic loop iteration {iteration}/{config.max_iterations}")

        # Call LLM with tools
        response = query_ollama_with_tools(
            messages, tools, model, api_url, config.timeout
        )

        assistant_msg = response.get("message", {})
        messages.append(assistant_msg)

        # Check for tool calls
        tool_calls = assistant_msg.get("tool_calls", [])

        if not tool_calls:
            # No tool calls - LLM is done
            return AgentLoopResult(
                content=assistant_msg.get("content", ""),
                tool_calls_made=tool_calls_made,
                iterations=iteration,
                stopped_reason="complete"
            )

        # Process each tool call
        for tc in tool_calls:
            func = tc.get("function", {})
            tool_name = func.get("name", "")
            tool_args = func.get("arguments", {})

            # Parse arguments if string
            if isinstance(tool_args, str):
                tool_args = json.loads(tool_args)

            parsed_call = ToolCall(name=tool_name, arguments=tool_args)
            tool_calls_made.append(parsed_call)

            if config.verbose:
                logger.info(f"Tool call: {tool_name}({tool_args})")

            # Execute tool with error handling
            try:
                result = tool_executor(tool_name, tool_args)
            except Exception as e:
                result = f"Error executing tool: {e}"
                logger.warning(f"Tool execution error: {e}")

            # Append tool result to messages
            messages.append({
                "role": "tool",
                "tool_name": tool_name,
                "content": result
            })

    # Max iterations reached
    logger.warning(f"Max iterations ({config.max_iterations}) reached")
    final_content = messages[-1].get("content", "") if messages else ""

    return AgentLoopResult(
        content=final_content + "\n\n[Note: Maximum iterations reached]",
        tool_calls_made=tool_calls_made,
        iterations=iteration,
        stopped_reason="max_iterations"
    )
```

### Tool Executor Pattern

```python
class ToolRegistry:
    """Registry for tool executors."""

    def __init__(self):
        self._tools: dict[str, Callable[[dict], str]] = {}
        self._schemas: list[dict] = []

    def register(self, name: str, schema: dict, executor: Callable[[dict], str]):
        """Register a tool with its schema and executor."""
        self._tools[name] = executor
        self._schemas.append(schema)

    def execute(self, name: str, arguments: dict) -> str:
        """Execute a tool by name."""
        if name not in self._tools:
            return f"Unknown tool: {name}"
        return self._tools[name](arguments)

    @property
    def schemas(self) -> list[dict]:
        return self._schemas
```

### Error Handling Strategy

Tool execution errors are caught and returned as the tool result, not raised:

```python
try:
    result = tool_executor(tool_name, tool_args)
except Exception as e:
    result = f"Error: {type(e).__name__}: {e}"
```

This allows the LLM to see the error and potentially try a different approach.
