### Agent YAML Schema

```yaml
# agents/scientist.yaml
name: scientist
description: "Science and research specialist"

corpus_filter:
  # Option 1: Explicit slug list
  slugs:
    - ap-biology-for-dummies
    - ap-chemistry-for-dummies
    - biochemistry-for-dummies

  # Option 2: Collection-based (can combine with slugs)
  collections:
    - science
    - research-methods

  # Optional: Exclude specific documents
  exclude_slugs:
    - solutions-manual
```

### Filter Logic

```python
def filter_manifest(manifest: ManifestData, config: AgentConfig) -> ManifestData:
    """Filter manifest to documents matching agent config.

    Matching logic:
    1. If slugs provided: include docs with matching slug
    2. If collections provided: include docs with any matching collection
    3. If both: union of slug matches and collection matches
    4. If neither: include all docs (no filter)
    5. Apply exclude_slugs to remove specific docs

    Returns:
        New ManifestData with filtered docs list
    """
```

### Filtering Examples

**Slug-only filter:**
```yaml
corpus_filter:
  slugs:
    - book-a
    - book-b
```
Result: Only `book-a` and `book-b` included.

**Collection-only filter:**
```yaml
corpus_filter:
  collections:
    - science
```
Result: All docs with `science` in their collections.

**Combined filter (union):**
```yaml
corpus_filter:
  slugs:
    - special-book
  collections:
    - science
```
Result: `special-book` PLUS all docs with `science` collection.

**With exclusions:**
```yaml
corpus_filter:
  collections:
    - science
  exclude_slugs:
    - solutions-manual
```
Result: All science docs EXCEPT `solutions-manual`.
