**I want** agent project context documents synced via Syncthing with symlink integration,

### CLAUDE.md Section to Add

```markdown
## Shared Context Architecture

Agent project context documents are stored in a Syncthing-shared folder for multi-machine access.

### Structure
- `~/Documents/Shared/Context/` - Agent working context (symlinked to `context/`)
- `~/Documents/Shared/Planning/` - Collaborative planning documents

### New Machine Setup
1. Install Syncthing and sync `~/Documents/Shared/` folder
2. Create symlink: `ln -s ~/Documents/Shared/Context ~/Documents/grounding-ai/context`
3. Verify: `ls context/` shows agent folders (ceo/, ip/, etc.)

### Troubleshooting
- **Symlink broken:** Recreate with `ln -s` command above
- **Git shows changes:** Ensure `.gitignore` includes `context/`
- **Sync conflicts:** Check Syncthing UI for `.sync-conflict` files
```

### Cross-Platform Notes

| Platform | Symlink Command | Notes |
|----------|-----------------|-------|
| macOS/Linux | `ln -s <target> <link>` | Native support |
| Windows | `mklink /D <link> <target>` | Requires admin or Developer Mode |
