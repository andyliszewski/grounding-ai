**Epic:** Epic 10 - Centralized Corpus Architecture
**Status:** Done

## Story

**As a** corpus query system,
**I want** to generate embeddings for only the documents an agent can access,
**So that** each agent has a focused vector index for efficient semantic search.

---

## Acceptance Criteria

1. `pdf2llm embeddings` subcommand extended with `--agent` flag
2. Agent config loaded from `agents/` directory
3. Manifest filtered before embedding generation
4. Only filtered documents embedded
5. Embeddings stored in agent-specific directory
6. `--check` flag compares timestamps and reports staleness
7. Exit code 1 if embeddings are stale (for scripting)
8. Clear progress output showing which agent and doc count
9. Works without `--agent` flag (full corpus, backward compatible)

### Staleness Check Logic

```python
def check_embeddings_staleness(
    corpus_path: Path,
    embeddings_path: Path,
    agent_config: AgentConfig | None = None
) -> tuple[bool, str]:
    """Check if embeddings are stale compared to corpus.

    Returns:
        (is_stale, reason) - True if embeddings need regeneration
    """
    # Load manifest
    manifest = ManifestManager.load(corpus_path / "_index.json")

    # Filter if agent provided
    if agent_config:
        manifest = filter_manifest(manifest, agent_config)

    # Check timestamp file
    timestamp_file = embeddings_path / ".timestamp"
    if not timestamp_file.exists():
        return True, "No timestamp file found"

    embedding_time = datetime.fromisoformat(timestamp_file.read_text().strip())
    manifest_time = datetime.fromisoformat(manifest.updated_utc)

    if manifest_time > embedding_time:
        return True, f"Corpus updated ({manifest.updated_utc}) after embeddings ({embedding_time})"

    return False, "Embeddings are up to date"
```

def test_agent_filtered_embeddings(tmp_path, sample_corpus_with_collections):
    """Test complete embedding generation with agent filter."""
    # Create agent config
    agents_dir = tmp_path / "agents"
    agents_dir.mkdir()
    (agents_dir / "scientist.yaml").write_text("""
name: scientist
corpus_filter:
  collections:
    - science
""")

    # Generate embeddings
    result = runner.invoke(app, [
        "embeddings",
        "--agent", "scientist",
        "--corpus", str(sample_corpus_with_collections),
        "--agents-dir", str(agents_dir),
        "--out", str(tmp_path / "embeddings" / "scientist")
    ])
    assert result.exit_code == 0

    # Verify output
    assert (tmp_path / "embeddings" / "scientist" / "index.faiss").exists()
    assert (tmp_path / "embeddings" / "scientist" / ".timestamp").exists()
